Design an intuitive and modern User Interface (UI) for an AI-powered Supply Chain Disruption Management System that integrates multiple intelligent agents such as Researcher Agent, Analyst Agent, and Negotiator Agent. The UI should allow supply chain managers to easily monitor global disruptions, analyze risks, and make faster operational decisions.

The dashboard should serve as the central interface where users can view real-time disruption alerts, supply chain status, and predictive insights generated by the AI agents. A clean layout with clearly defined sections should display information such as affected regions, severity levels, estimated financial impact, and recommended actions.

The UI must include a dynamic visualization panel showing maps, charts, and graphs that highlight disruptions like port delays, labor strikes, natural disasters, or geopolitical events. Color-coded indicators (green, yellow, red) should help users quickly identify the urgency of each disruption.

Another key UI component should be the Agent Interaction Panel, where users can view outputs from different agents. The Researcher Agent provides detected global disruption signals, the Analyst Agent presents impact analysis and risk scores, and the Negotiator Agent suggests alternative suppliers, routes, or mitigation strategies. Users should also be able to filter information based on region, product category, or time.

The interface should include interactive features such as search, filtering, notification alerts, and customizable dashboards. Managers should receive automated recommendations but still have the ability to manually review and approve decisions.

The UI design should prioritize simplicity, responsiveness, and usability, ensuring that even complex supply chain data is presented in a clear and understandable manner. Mobile compatibility and role-based access should also be considered so that different stakeholders can access relevant insights securely.

Overall, the UI should enable faster decision-making, improve supply chain visibility, and enhance collaboration between human managers and AI agents.
